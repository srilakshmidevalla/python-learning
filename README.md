🐍 Python for Beginners
A structured, hands-on Python learning repo. Each folder = one concept. Each file = one runnable example.
📁 Folder Structure
python-beginner-repo/
├── 01_basics/              # Variables, data types, input/output
├── 02_control_flow/        # if/else, loops
├── 03_functions/           # def, return, scope
├── 04_data_structures/     # lists, dicts, tuples, sets
├── 05_file_handling/       # read/write files
├── 06_mini_projects/       # small real-world projects
└── exercises/              # practice problems with solutions
🚀 How to Run
bash# Clone the repo
git clone https://github.com/YOUR_USERNAME/python-beginner-repo.git
cd python-beginner-repo

# Run any file
python 01_basics/01_hello_world.py
